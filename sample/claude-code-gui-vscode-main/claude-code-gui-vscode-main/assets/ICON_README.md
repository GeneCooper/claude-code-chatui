# Icon Assets

## icon.svg

A placeholder SVG icon for the extension. For marketplace publishing,
convert this to a 128x128 PNG file named icon.png.

To convert:

1. Open icon.svg in a browser or image editor
2. Export as PNG at 128x128 pixels
3. Save as icon.png in this directory

Or use a tool like:

- npx svgexport icon.svg icon.png 128:128
- ImageMagick: convert icon.svg -resize 128x128 icon.png
